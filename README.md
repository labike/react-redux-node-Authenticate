# react-redux-node-Authenticate

#### backend
    * express
    * mongodb
    * bcrypt hash password
    * 'JsonWebToken' token authenticate

#### frontend
    * react
    * redux
    * react-router
    
#### start

backend start

```
mongod
cd backend
npm install
nodemon app.js
```

front start

```
cd frontend
yarn install
yarn start
```
